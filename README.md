## 1. 🥇 top (시스템 실시간 모니터링)

`top` 명령어는 시스템의 **실시간 상태**와 실행 중인 **프로세스 활동**을 동적으로 확인하는 데 사용됩니다. CPU, 메모리, 스왑, 로드 에버리지 등의 시스템 자원 사용 현황을 즉각적으로 보여줍니다.

### 📊 주요 모니터링 정보

| 섹션 | 표시 내용 | 역할 |
| :---: | :--- | :--- |
| **Header** | 시스템 시간, 부팅 시간, 로드 에버리지 | 시스템의 전반적인 부하 수준 |
| **Task** | 총 프로세스 수, 실행/슬립/정지된 프로세스 수 | 현재 시스템에서 처리되는 작업의 상태 |
| **CPU** | 사용자, 시스템, 유휴 시간 비율 (User, System, Idle) | CPU 자원 할당 및 사용 효율 |
| **Mem** | 물리적 메모리(RAM) 및 스왑 메모리 사용량 | 메모리 확보 및 부족 여부 판단 |

### 💻 사용 예시

```bash
# 기본 사용법: 실시간 모니터링 시작
top

# 특정 사용자(user)의 프로세스만 보기 (top 실행 후 'u' 입력)
top -u [username]
